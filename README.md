# Mr.Camel 기업과제

## 요구사항

고객들이 원하는 상품 목록에 대한 조회 이력 페이지를 구현합니다.

- 상품 조회 이력 관리
- 조회 이력에 필터 적용
  - 브랜드 명
  - 관심없는 상품
  - 최근 조회 순
  - 낮은 가격 순
- 상세 페이지 랜덤 로드 기능
- 관심없는 상품 클릭 시 경고메세지 노출
